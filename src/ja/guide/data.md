# {{$frontmatter.title}}

この拡張機能は、ブラウザが提供する`chrome.storage.local` APIを使用してデータをローカルにのみ保存し、データをどこにも送信しません。

しかし、データを転送するのに役立つツールをサポートしています。

## エクスポートとインポート

バックエンドページでデータを`.json`ファイルにエクスポートし、他のブラウザでインポートできます。この操作は、新しいデバイスや新しいブラウザを使い始めたばかりの場合に適しています。

また、他の類似の拡張機能からエクスポートされたデータをインポートし、ローカルデータを上書きするか追加するかを決定できます。

| 他の拡張機能              | ファイル形式        | 備考                             |
| ------------------------- | ------------------- | -------------------------------- |
| Webtime Tracker           | `.csv`または`.json` |                                  |
| Web Activity Time Tracker | `.csv`              |                                  |
| History Trends Unlimited  | `.tsc`              | 訪問回数のデータのみが含まれます |

## クライアント間でデータを同期する

この拡張機能は、データをリモートで保存するサービスをサポートしていません。

代わりに、信頼できるサードパーティサービスにデータを同期するのに役立つツールを提供しています。

| サードパーティサービス | 前提条件                                                                                                                                   |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Github Gist            | Gist権限を付与するGithubアクセストークン                                                                                                   |
| Obsidian               | プラグイン[Local REST API for Obsidian](https://github.com/coddingtonbear/obsidian-local-rest-api)がインストールされたObsidianクライアント |

トークンが正しく設定されている場合、記録ページで他のクライアントのデータをクエリできます。
