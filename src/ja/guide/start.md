# {{$frontmatter.title}}

わずか3つの簡単なステップで、この拡張機能の使用を開始できます。

## 1. アイコンをピン留めする

まず、この拡張機能をより便利に使用するために、ピン留めすることをお勧めします。これを行う方法はブラウザによって異なります。以下の図はChromeでの方法を示しています。

<div style="display: flex; justify-content: center;">
    <img src="/images/pin.png" width=400 />
</div>

## 2. 任意のウェブサイトを閲覧する

次に、任意のウェブサイトを閲覧すると、アイコンに時間が刻まれているのが見えます。このように <img src="/images/beating.gif" style="display: inline-block;height: 24px;"/>。

## 3. ポップアップページでデータを読む

最後に、アイコンをクリックしてポップアップページを開くと、今日、今週、または今月の円グラフで視覚化されたデータを読み取ることができます。

このページの下部にあるリンクをクリックすると、すべての機能にアクセスできます。

::: tip
基本的な使い方を学びました。試してみてください！
:::
