# {{$frontmatter.title}}

基本レコードに加えて、バックエンドページで分析レポートを表示することもできます。これには、習慣レポートとサイトレポートが含まれます。

## 習慣レポート

習慣レポートは2つの部分で構成されています。

-   ウェブサイト使用習慣。この部分は、最も訪問されたコンテンツを示します。
-   期間習慣。この部分は、最も忙しい時間と最も長いアイドル期間を示します。

## サイトレポート

同様に、サイトレポートも2つの部分で構成されています。

-   概要。この部分は、選択したサイトの概要データを示します。
-   トレンド。この部分は、選択したウェブサイトの1日あたりの閲覧時間と訪問回数の履歴トレンドを示します。
