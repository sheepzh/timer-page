# {{$frontmatter.title}}

上記の分析レポートを通じて、役に立たないウェブサイトにどれだけの時間を費やしているかを認識したら、時間を制御する必要があるかもしれません。

この拡張機能を使用すると、任意のウェブサイトでの行動を制限できます。

## 制限ルールを作成する

まず、バックエンドページでルールを作成する必要があります。

各ルールには、次の表に示すようにいくつかの部分が含まれています。

| 部分             | 説明                                                                                                   |
| ---------------- | ------------------------------------------------------------------------------------------------------ |
| 制限されたURL    | 制限する必要があるURL。式がサポートされています。例：`https://www.github.com`、`*://www.google.com/**` |
| 1日の制限        | 1日あたりの最大時間。ゼロは制限なしを意味します。                                                      |
| 訪問ごとの制限   | 訪問ごとの最大時間。ゼロは制限なしを意味します。                                                       |
| 許可されない期間 | ターゲットURLへのアクセスが常に制限される期間。                                                        |

## 制限された

いずれかのルールの制限を満たすと、訪問しているページは閲覧を妨げるマスクで覆われます。

続行するには、ルールページに移動して、このルールを無効にするか変更する必要があります。オプションでロック解除操作が設定されている場合、確認またはパスワードも必要です。

::: warning
不快な確認コードは本当に難しく不快です。ロックを解除するのに時間がかかりすぎる可能性があります！
:::
