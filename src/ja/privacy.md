# {{$frontmatter.title}}

::: info
完全なサービスを提供するため、この拡張機能は使用中に必然的にいくつかの個人データを収集します。詳細については、以下のプライバシーポリシーをご覧ください。
:::

## どのようなデータが収集されますか？

| 内容                 | 使用目的                                                                                                   | 必須                                           |
| -------------------- | ---------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| ウェブサイト閲覧履歴 | 閲覧時間と訪問回数のカウントに使用                                                                         | はい                                           |
| タブ情報             | ウェブサイトの名前とアイコンを自動的に取得し、データ表示時のユーザーエクスペリエンスを向上させるために使用 | オプションでこの機能が有効になっている場合のみ |

## これらのデータはどう扱われますか？

この拡張機能によって収集されたすべてのデータは、ブラウザのローカルストレージにのみ保存され、他の場所に配布されることはありません。

ただし、拡張機能が提供するツールを使用して、JSONまたはCSVファイル形式でデータをエクスポートまたはインポートできます。また、GitHub Gistなど、十分に信頼できるサードパーティサービスを使用してデータをバックアップすることもできます。

::: tip
私たちはデータの収集をお手伝いするだけで、処分権はあなたのものです。
:::
