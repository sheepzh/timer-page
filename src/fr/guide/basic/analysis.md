# {{$frontmatter.title}}

En plus des enregistrements de base, vous pouvez également voir des rapports analytiques sur la page backend, y compris des rapports d'habitudes et des rapports de site.

## Rapport d'habitudes

Le rapport d'habitudes se compose de deux parties.

-   Habitudes d'utilisation du site Web. Cette partie montre votre contenu le plus visité.
-   Habitudes de période. Cette partie montre vos périodes les plus occupées et les périodes d'inactivité les plus longues.

## Rapport de site

De même, le rapport de site se compose également de deux parties.

-   Résumé. Cette partie montre les données de présentation pour le site sélectionné.
-   Tendances. Cette partie montre la tendance historique du temps de visualisation et du nombre de visites par jour pour le site Web sélectionné.
