# {{$frontmatter.title}}

::: info
Afin de vous fournir des services complets, cette extension collectera nécessairement certaines de vos données personnelles pendant l'utilisation, voir la déclaration de confidentialité suivante pour plus de détails.
:::

## Quelles données sont collectées ?

| Contenu                      | Utilisation                                                                                                                              | Obligatoire                                               |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------- |
| Historique de navigation     | Utilisé pour compter le temps de navigation et les visites                                                                               | Oui                                                       |
| Informations sur les onglets | Utilisé pour obtenir automatiquement le nom et l'icône du site Web et améliorer l'expérience utilisateur lors de l'affichage des données | Uniquement si cette fonction est activée dans les options |

## Que faire avec ces données ?

Nous garantissons que toutes les données collectées par cette extension ne seront enregistrées que dans le stockage local de votre navigateur et ne seront jamais distribuées ailleurs.

Vous pouvez cependant utiliser les outils fournis par l'extension pour exporter ou importer vos données au format de fichier JSON ou CSV. L'extension vous permet également d'utiliser GitHub Gist, etc., des services tiers en qui vous avez suffisamment confiance pour sauvegarder vos données.

::: tip
Nous ne faisons que vous aider à collecter des données, mais le droit de disposition doit être le vôtre.
:::
