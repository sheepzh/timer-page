# {{$frontmatter.title}}

::: info
Para proporcionarte servicios completos, esta extensión necesariamente recopilará algunos de tus datos personales durante el uso, consulta la siguiente declaración de privacidad para obtener más detalles.
:::

## ¿Qué datos se recopilan?

| Contenido               | Uso                                                                                                                              | Requerido                                            |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| Historial de navegación | Se utiliza para contar el tiempo de navegación y las visitas                                                                     | Sí                                                   |
| Información de pestañas | Se utiliza para obtener automáticamente el nombre y el icono del sitio web y mejorar la experiencia del usuario al mostrar datos | Solo si esta función está habilitada en las opciones |

## ¿Qué hacer con estos datos?

Garantizamos que todos los datos recopilados por esta extensión solo se guardarán en el almacenamiento local de tu navegador y nunca se distribuirán en otro lugar.

Sin embargo, puedes usar las herramientas proporcionadas por la extensión para exportar o importar tus datos en formato de archivo JSON o CSV. La extensión también te permite usar GitHub Gist, etc., servicios de terceros en los que confíes lo suficiente para respaldar tus datos.

::: tip
Solo te ayudamos a recopilar datos, pero el derecho de disposición debe ser tuyo.
:::
