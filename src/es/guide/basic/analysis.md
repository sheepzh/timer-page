Además de los registros básicos, también puedes ver informes analíticos en la página de backend, incluidos informes de hábitos e informes de sitios.

## Informe de hábitos

El informe de hábitos consta de dos partes.

-   Hábitos de uso del sitio web. Esta parte muestra tu contenido más visitado.
-   Hábitos de período de tiempo. Esta parte muestra tus momentos más ocupados y períodos de inactividad más largos.

## Informe de sitio

Del mismo modo, el informe de sitio también consta de dos partes.

-   Resumen. Esta parte muestra datos generales para el sitio seleccionado.
-   Tendencias. Esta parte muestra la tendencia histórica del tiempo de visualización y el número de visitas por día para el sitio web seleccionado.
