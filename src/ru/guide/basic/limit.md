# {{$frontmatter.title}}

Как только вы осознаете, сколько времени вы тратите на бесполезные веб-сайты через приведенный выше отчет об анализе. Возможно, вам нужно контролировать время.

Это расширение позволяет ограничивать ваше поведение на любом веб-сайте.

## Создание правил ограничения

Во-первых, вам нужно создать правила на странице бэкенда.

Каждое правило содержит несколько частей, как показано в следующей таблице.

| Часть                    | Описание                                                                                                                  |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------- |
| Ограниченный URL         | URL, который необходимо ограничить. Поддерживаются выражения. Например, `https://www.github.com`, `*://www.google.com/**` |
| Ежедневное ограничение   | Максимальное время в день, ноль означает отсутствие ограничения.                                                          |
| Ограничение на посещение | Максимальное время на посещение, ноль означает отсутствие ограничения.                                                    |
| Запрещенные периоды      | Периоды времени, в течение которых доступ к целевому URL всегда ограничен.                                                |

## Ограничено

Если вы соответствуете ограничениям любого правила, страницы, которые вы посещаете, будут покрыты маской, которая не позволяет вам просматривать.

Чтобы продолжить, вы должны перейти на страницу правил и отключить или изменить это правило. Если операция разблокировки установлена в опции, также требуется проверка или пароль.

::: warning
Отвратительный проверочный код действительно сложен и отвратителен, это может занять слишком много времени, чтобы разблокировать его!
:::
