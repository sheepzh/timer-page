# {{$frontmatter.title}}

Это расширение хранит ваши данные только локально с помощью API `chrome.storage.local`, предоставляемого браузером, и никогда не отправляет ваши данные куда-либо.

Но оно поддерживает инструменты, которые помогают вам передавать ваши данные.

## Экспорт и импорт

Вы можете экспортировать данные в файл `.json` на странице бэкенда, а затем импортировать их в другие браузеры. Следует отметить, что эта операция более подходит, если вы только начинаете использовать новое устройство или новый браузер.

Вы также можете импортировать данные, экспортированные из других подобных расширений, и решить, перезаписать или добавить к локальным данным.

| Другое расширение         | Формат файла       | Примечание                                    |
| ------------------------- | ------------------ | --------------------------------------------- |
| Webtime Tracker           | `.csv` или `.json` |                                               |
| Web Activity Time Tracker | `.csv`             |                                               |
| History Trends Unlimited  | `.tsc`             | Содержит только данные о количестве посещений |

## Синхронизация данных между клиентами

Это расширение не поддерживает службы, которые хранят данные удаленно.

Вместо этого оно предоставляет инструменты, которые помогают вам синхронизировать данные в сторонних службах, которым вы доверяете.

| Сторонняя служба | Требования                                                                                                                        |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| Github Gist      | Токен доступа Github с правами Gist                                                                                               |
| Obsidian         | Клиент Obsidian с установленным плагином [Local REST API for Obsidian](https://github.com/coddingtonbear/obsidian-local-rest-api) |

Если токен настроен правильно, вы можете запросить данные других клиентов на странице записи.
