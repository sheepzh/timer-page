# {{$frontmatter.title}}

Você pode começar rapidamente a usar esta extensão em apenas 3 etapas fáceis.

## 1. Fixar o ícone

Primeiramente, para usar esta extensão de forma mais conveniente, é melhor fixá-la. Não é o mesmo em diferentes navegadores para fazer isso, a figura a seguir mostra como no Chrome.

<div style="display: flex; justify-content: center;">
    <img src="/images/pin.png" width=400 />
</div>

## 2. Navegar em qualquer site

Em seguida, navegue em qualquer site, e você verá que o tempo está batendo no ícone, assim <img src="/images/beating.gif" style="display: inline-block;height: 24px;"/>.

## 3. Ler dados na página popup

Finalmente, clique no ícone para abrir a página popup, e você poderá ler os dados visualizados com gráfico de pizza, de hoje, desta semana ou deste mês.

E você pode acessar todos os recursos clicando no link na parte inferior desta página.

::: tip
Você aprendeu o uso básico, experimente!
:::
